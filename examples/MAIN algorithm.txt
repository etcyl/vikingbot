Set constants, import modules, do all initialization here

MAIN: 
1. Check front, left, and right for obstacles
   1. Return learner classification
      1. Class 1: no objects left, right, or front
      2. Class 2: 1 object front
      3. Class 3: 1 object left
      4. Class 4: 1 object right
      5. Class 5: objects left, right, and front
   1. emote_and_drive(classifier label)
      1. If c_label == 1:
         1. drive()
         2. Return to MAIN
      1. Else if c_label == 2:
         1. emote(front)
         2. drive(left, right); #Move left or right a little bit
         3. Return to MAIN
      1. Else if c_label == 3:
         1. emote(left)
         2. drive(front, right)
         3. Return to MAIN
      1. Else if c_label == 4:
         1. emote(right)
         2. drive(front, left)
         3. Return to MAIN
      1. Else if c_label == 5:
         1. emote(all);
         2. drive(behind)
         3. Return to MAIN
